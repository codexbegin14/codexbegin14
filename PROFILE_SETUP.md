# GitHub Profile Setup Checklist

Use this checklist to make the `codexbegin14` GitHub profile look professional and recruiter-friendly.

## 1. Create the Special Profile Repository

Create a new public repository with this exact name:

```text
codexbegin14
```

Important settings:

- Owner: `codexbegin14`
- Repository name: `codexbegin14`
- Visibility: Public
- Add README: No

After creating it, upload or push the `README.md` from this folder. GitHub will automatically show it at the top of your profile.

## 2. Update Profile Details

Go to:

```text
GitHub profile -> Edit profile
```

Use these values:

```text
Name:
Muhammad Awais Shaikh

Bio:
CS student at FAST NUCES | Full-stack developer building clean web systems and practical software products

Location:
Karachi, Pakistan

Website:
https://muhammad-awais.vercel.app
```

Recommended profile photo:

- Use the same professional photo from your portfolio.
- Avoid cartoon/anime avatars for HR-facing profiles.
- Keep the face centered and clear.

## 3. Pin These Repositories

Go to:

```text
Profile -> Customize your pins
```

Recommended pins:

1. `muhammad-awais-portfolio`
2. `FORKED-NUCES`
3. `Guilty-by-Data`
4. `Investment-Recommendation-System`
5. `Hybrid_Load_Balancer`
6. One more clean, complete project only if it has a good README

Avoid pinning:

- Empty repositories
- Small calculator/test repos
- Forks unless they are genuinely important
- Repositories without README files

## 4. Improve Repository Descriptions

Add short descriptions to important repos:

```text
muhammad-awais-portfolio
Modern full-stack developer portfolio built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

FORKED-NUCES
FAST-NUCES student collaboration platform for open-source university projects and skill-matched contributors.

Guilty-by-Data
Investigative competition platform with timed rounds, participant login, leaderboards, and admin workflows.

Investment-Recommendation-System
Stock and fund recommendation system using Python, Pandas, Scikit-learn, and filtering techniques.
```

## 5. Optional Cleanup

For old beginner repos like `cal`, `calculator`, or empty game repos:

- Make them private, or
- Add a basic README, or
- Unpin them from the profile

This keeps the first impression focused on your strongest work.
